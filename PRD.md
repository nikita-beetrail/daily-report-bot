# Бот для ежедневных отчётов

## Цель
Веб-приложение для ежедневной записи рабочих событий и автоматической отправки отчёта руководителю в Telegram.

## Пользователи
- **Я (руководитель проектов)** — заполняю отчёт за день
- **Мой руководитель** — получает готовый отчёт в Telegram

## Основной сценарий
1. Открываю приложение (с телефона или компьютера)
2. Выбираю дату (по умолчанию — сегодня)
3. Выбираю проект из списка
4. Добавляю выполненные задачи с указанием статуса
5. Описываю проблемы/блокеры (если есть)
6. Пишу план на завтра
7. Нажимаю «Сохранить черновик» или «Отправить отчёт»
8. Отчёт форматируется и отправляется в Telegram руководителю

## Функционал

### Форма отчёта
- Дата (по умолчанию сегодня)
- Выпадающий список проектов
- Динамический список задач (можно добавлять/удалять)
- Статус каждой задачи: ✅ выполнено, 🔄 в процессе, 🚫 заблокировано
- Поле «Проблемы и блокеры» (текст)
- Поле «План на завтра» (текст)
- Поле «Комментарий» (необязательное)

### Отправка в Telegram
- Отчёт отправляется через Telegram Bot API
- Формат сообщения: структурированный текст с эмодзи
- Получатель: chat_id руководителя

### История отчётов
- Список всех отправленных отчётов
- Возможность просмотреть любой прошлый отчёт
- Фильтр по дате и проекту

### Черновики
- Автосохранение каждые 30 секунд
- Кнопка «Сохранить черновик»
- Возможность вернуться к черновику позже

## Формат сообщения в Telegram

📅 Дата: 07.04.2026
📁 Проект: [название]

**Выполненные задачи:**
✅ Задача 1
🔄 Задача 2 (в процессе)
🚫 Задача 3 (заблокировано)

Текст проблемы

Текст плана

Текст комментария
