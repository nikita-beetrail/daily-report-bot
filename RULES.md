# Правила проекта

## Общие
- Интерфейс полностью на русском языке
- Комментарии в коде на русском
- Не удаляй существующий код без явного запроса
- Каждое изменение — маленький шаг, не переписывай всё сразу

## Дизайн
- Минималистичный, чистый интерфейс
- Тёмная тема по умолчанию (с возможностью переключения на светлую)
- Адаптивный дизайн: удобно и на телефоне, и на компьютере
- Скруглённые углы у карточек (border-radius: 12px)
- Приятные отступы, не перегружать экран

## Код
- Чистый, читаемый код
- Компоненты — маленькие и переиспользуемые
- Обработка ошибок: показывать понятные сообщения пользователю
- Валидация формы перед отправкой
- Не отправлять пустой отчёт

## Telegram интеграция
- Использовать Telegram Bot API (https://api.telegram.org)
- Токен бота и chat_id хранить в переменных окружения (.env)
- Никогда не хардкодить токены в коде
- При ошибке отправки — показать сообщение и сохранить как черновик
