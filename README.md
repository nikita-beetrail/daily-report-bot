# Daily report bot — UI

Пока это только интерфейс (без Supabase и Telegram).

## Локальный запуск

Установите Node.js 18+.

Затем:

```bash
npm i
npm run dev
```

## Что уже есть

- React + TypeScript (Vite)
- Tailwind CSS
- React Router
- React Hook Form
- Главная страница `/` — форма ежедневного отчёта
- Тёмная тема по умолчанию, переключатель темы
- Автосохранение черновика в `localStorage` каждые 30 секунд
